# 🌍 WanderWorld - Travel Website

## 📌 Project Overview

WanderWorld is a modern responsive travel website designed to help users explore beautiful destinations, view travel packages, and submit booking enquiries.

This project was developed as a frontend website assignment using HTML5, CSS3, and JavaScript with SEO optimization and responsive design.

---

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Modern travel website UI
- Hero banner with travel theme
- Popular destination cards
- Travel packages section
- Image gallery with captions
- Booking enquiry form
- JavaScript form validation
- Mobile navigation menu
- Automatic hero image slider
- SEO optimized pages
